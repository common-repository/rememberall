Daily email report if updates are available.

== Description ==
Themes, plugins and the WordPress core need to be updated. WordPress updates help keep your website secure and bug-free, ensuring you have the latest features, better compatibility, and a smooth WordPress experience. Rememberall helps you remember these updates. As soon as there are updates, the site admin will receive an email notification.


== Installation ==


1. Upload the entire `rememberall` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Finished. The admin gets an email once a day, if an update is available.
